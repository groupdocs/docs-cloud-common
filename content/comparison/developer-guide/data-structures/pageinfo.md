## PageInfo ##

PageInfo data structure represents information about page

PageInfo example:


```javascript 

{
  "Width": 0,
  "Height": 0,
  "PageNumber": 0
}

 ```



 

|Name|Description
|---|---
|Width|Page width
|Height|Page height
|PageNumber|Page number
