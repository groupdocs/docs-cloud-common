## Link ##

Link data structure represents output document information

Link example:


```javascript 

{
  "href": "string",
  "rel": "string",
  "type": "string",
  "title": "string"
}

 ```



 

|Name|Description
|---|---
|Href|Contains the link's URL
|Rel|Contains file path in the cloud storage
|Type|Output type ('file')
|Title|File name
