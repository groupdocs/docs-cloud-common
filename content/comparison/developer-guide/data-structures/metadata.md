## Metadata ##

Metadata data structure used for setting output document metadata.

Metadata example:


```javascript 

{
  "Author": "string",
  "LastSaveBy": "string",
  "Company": "string"
}

 ```



 

|Name|Description
|---|---
|Author|Value of document Author
|LastSaveBy|Value of last save by author of document
|Company|Value of Company of document
